# Backlog

This folder holds ideas that matter but are not active yet.

Use it like this:

- `roadmap.md` explains the current direction.
- `tasks.md` tracks active epics and work in flight.
- `backlog/ideas.md` stores small ideas, experiments, and one-off notes.
- `backlog/epics.md` stores larger themes that are not ready to move into active work.

When something becomes real work, move it from `backlog/` into `tasks.md`.
